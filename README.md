# GitHub User Finder

A simple **JavaScript web app** that fetches GitHub user information and repositories using the **GitHub API**. It also allows users to create a new repository using a **GitHub Personal Access Token**.

## Features

* Search any GitHub username
* View followers, following, and public repositories
* Display user repositories with clickable links
* Create a new public repository

## Technologies Used

* HTML
* JavaScript
* Tailwind CSS
* GitHub REST API

## API Endpoints Used

* `https://api.github.com/users/{username}`
* `https://api.github.com/users/{username}/repos`
* `https://api.github.com/user/repos`

## How to Use

1. Enter a GitHub username to fetch user details and repositories.
2. To create a repository, enter your **GitHub token** and **repository name**.
3. Click **Create Repository** to create a new repo.
